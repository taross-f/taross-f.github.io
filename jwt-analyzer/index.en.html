üí° About JWT

JWT (JSON Web Token) is a compact, URL-safe means of representing claims to be transferred between two parties. It consists of three parts separated by dots (.): Header, Payload, and Signature.

  • Header: Contains metadata about the token (algorithm, type)
  • Payload: Contains the claims (user data, permissions, expiration)
  • Signature: Used to verify the token hasn't been tampered with

Note: This tool only decodes and displays JWT contents. It does not verify signatures.
