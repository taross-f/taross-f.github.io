💡 JSONについて

JSON（JavaScript Object Notation）は、軽量なデータ交換フォーマットです。主な用途：

  • Web APIでのデータ交換
  • 設定ファイルの保存
  • データベースでの構造化データ保存
  • JavaScriptでのオブジェクト表現

ショートカット: Ctrl+Enter で整形、Ctrl+Shift+Enter で圧縮
