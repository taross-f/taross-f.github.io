# 修正点

* Topの記事一覧で日付出てほしかったので、下の方に追加。合わせて日付の書式も日本っぽく? YYYY.MM.DDに変更

index.html
```html
      <div class="date">
          \{\{ post.date | date: "Posted on %Y.%m.%d" \}\}
      </div>

```

* 日付の文字がデカかったので修正。ついでになんとなく右寄せに

style.scss
```scss
 .date {
    font-style: italic;
    color: $gray;
    float: right;
    font-family: $helveticaNeue;
    font-size: 18px;
 
  }

```

* post側の書式も合わせて修正

post.html
```html
   <div class="date">
      \{\{ page.date | date: "Posted on %Y.%m.%d" \}\}
   </div>

```
